# Wedding Invitation - Ajinkya & Shalini

A luxury Royal Rajasthani digital wedding invitation with real-time admin dashboard.

## 🚀 Features
- **Mandatory Lead Catcher** — every guest logs in before seeing the invitation
- **Interactive 3D Loader** — 8-second immersive entrance animation  
- **Live Countdown** — to April 26, 2026
- **Add to Google Calendar** — all 7 events in one click
- **Full Timeline** — Mehendi, Haldi, Sangeet, Pheras & more
- **RSVP System** — with Firebase real-time storage
- **Admin Dashboard** — real-time guest activity tracking via Firebase

## 🔧 Setup

```bash
npm install
cp .env.example .env
# Fill in your Firebase config in .env
npm run dev
```

## 🔥 Firebase Setup
Set these in `.env`:
```
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_DATABASE_URL=...
VITE_FIREBASE_PROJECT_ID=...
VITE_FIREBASE_APP_ID=...
```
